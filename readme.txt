This is an online quick response code payement application  integrated with arental mangement system with two users, the admin and the student.
To get started, make sure you have cloned the source code ,have composer and stripe installed.
Download or set up any local web server that runs PHP script.
Open the web-server database and create a new database name it house_rental_db.
Import the SQL file located in the database folder of the source code.
Copy and paste the source code to the location where your local web server accessing your local projects. Example for XAMPP('C:\xampp\htdocs')
Open a web browser and browse the project. E.g [http://localhost/qr_house-rental-payment-system/mainpage]
